## 方式一：添加到主屏幕（推荐⭐）

### iPhone (iOS) 步骤：
1. **Safari浏览器**打开应用链接
2. 点击底部**分享按钮**（方块带箭头）
3. 选择**"添加到主屏幕"**
4. 点击**"添加"**
5. ✅ 桌面出现App图标，像原生App一样使用

### Android 步骤：
1. **Chrome浏览器**打开应用链接
2. 点击右上角**菜单**（三个点）
3. 选择**"添加到主屏幕"**或**"安装应用"**
4. 点击**"添加"**
5. ✅ 桌面出现App图标

### 效果：
- 🏠 独立图标，点击直接打开
- 📴 支持离线使用（已缓存）
- 🎨 全屏显示，无浏览器地址栏
- 💾 数据本地存储

## 方式二：局域网访问（本地演示）

### 步骤：

#### 1. 启动本地服务器
```bash
# 在项目目录下执行
cd pet-store-demo-app

# Python方式
python3 -m http.server 8080

# Node.js方式（需安装http-server）
npx http-server -p 8080

# PHP方式
php -S 0.0.0.0:8080
```

#### 2. 获取本机IP
```bash
# Mac/Linux
ifconfig | grep "inet " | grep -v 127.0.0.1

# Windows
ipconfig
```

#### 3. 生成二维码
```bash
# 使用生成的脚本
python3 generate_qr.py http://你的IP:8080

# 或者手动替换下面的IP
# 例如: http://192.168.1.100:8080
```

#### 4. 手机扫码访问
- 手机连接到**同一WiFi**
- 用微信/浏览器**扫描二维码**
- 立即访问应用

## 方式三：公网访问（给客户演示）

### 方案A：使用内网穿透（ngrok）

#### 1. 安装ngrok
```bash
# Mac
brew install ngrok

# Windows
# 下载: https://ngrok.com/download
```

#### 2. 注册并配置
```bash
ngrok authtoken 你的token
```

#### 3. 启动穿透
```bash
ngrok http 8080
```

#### 4. 获取公网链接
ngrok会显示：
```
Forwarding  http://xxxx.ngrok.io -> http://localhost:8080
```

#### 5. 生成二维码
```bash
python3 generate_qr.py http://xxxx.ngrok.io
```

✅ **客户在任何地方都能访问**

---

### 方案B：部署到GitHub Pages（免费永久）

#### 1. 创建GitHub仓库
- 访问 https://github.com/new
- 仓库名：`pet-store-demo`
- 选择 **Public**

#### 2. 上传文件
```bash
# 在pet-store-demo-app目录下
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/你的用户名/pet-store-demo.git
git push -u origin main
```

#### 3. 开启GitHub Pages
- 进入仓库 **Settings**
- 左侧点击 **Pages**
- Source选择 **Deploy from a branch**
- Branch选择 **main** / **root**
- 点击 **Save**

#### 4. 访问链接
等待2分钟后访问：
```
https://你的用户名.github.io/pet-store-demo/
```

#### 5. 生成二维码
```bash
python3 generate_qr.py https://你的用户名.github.io/pet-store-demo/
```

✅ **永久免费，全球可访问**

### 方案C：部署到Vercel（推荐）

#### 1. 安装Vercel CLI
```bash
npm i -g vercel
```

#### 2. 部署
```bash
cd pet-store-demo-app
vercel
```

#### 3. 按提示操作
- 登录Vercel账号
- 确认部署

#### 4. 获取链接
部署完成后显示：
```
🔍  Inspect: https://vercel.com/xxx/pet-store-demo
✅  Production: https://pet-store-demo-xxxxx.vercel.app
```

#### 5. 生成二维码
```bash
python3 generate_qr.py https://pet-store-demo-xxxxx.vercel.app
```

✅ **自动HTTPS，访问速度快**

## ❓ 常见问题

**Q: 手机打不开局域网链接？**
A: 确保手机和电脑在同一WiFi下

**Q: iPhone添加到主屏幕后白屏？**
A: 确保使用Safari，且网站支持HTTPS

**Q: Android提示"不安全"？**
A: 使用HTTPS链接，或部署到支持HTTPS的平台

**Q: 二维码扫不出来？**
A: 检查链接是否正确，网络是否通畅
