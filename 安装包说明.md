# 🐾 宠物管家 - 手机App安装包

## 📦 安装包内容

```
pet-store-demo-app/
├── index.html              # 主应用（单页面Web App）

├── qrcode-example.png     # 示例二维码
```

## 📱 安装方式

### ✅ 方式一：添加到主屏幕（最像原生App）

**iPhone安装步骤：**
1. 用Safari打开应用链接
2. 点击分享按钮
3. 选择"添加到主屏幕"
4. 完成！桌面出现App图标

**Android安装步骤：**
1. 用Chrome打开应用链接
2. 菜单 → "添加到主屏幕"
3. 完成！像普通App一样使用

**特点：**
- ✨ 独立图标，点击直接打开
- 📴 支持离线使用
- 🎨 全屏显示，无浏览器地址栏
- 💾 数据本地存储

---

### ✅ 方式二：扫码直接访问

![示例二维码](./qrcode-example.png)

**使用步骤：**
1. 将应用部署到服务器（见下方）
2. 生成二维码（已提供脚本）
3. 手机微信/浏览器扫码
4. 即可访问完整应用

**生成二维码命令：**
```bash
python3 generate_qr.py http://你的链接
```

## 🚀 部署方案（选择一种）

### 方案A：GitHub Pages（免费永久，推荐⭐）

```bash
# 1. 创建GitHub仓库
# 2. 上传文件
# 3. 开启GitHub Pages
# 4. 访问：https://用户名.github.io/仓库名/
```

**优点：**
- ✅ 完全免费
- ✅ 永久在线
- ✅ 自动HTTPS
- ✅ 全球可访问

---

### 方案B：Vercel（快速部署）

```bash
# 1. 安装vercel
npm i -g vercel

# 2. 部署
cd pet-store-demo-app
vercel

# 3. 获得链接：https://xxx.vercel.app
```

**优点：**
- ✅ 一键部署
- ✅ HTTPS支持
- ✅ 访问速度快

---

### 方案C：局域网演示（本地测试）

```bash
# 1. 启动服务器
python3 -m http.server 8080

# 2. 获取本机IP
ifconfig/ipconfig

# 3. 手机连同一WiFi
# 4. 访问：http://IP:8080
```

**适合：** 本地给客户演示

---

### 方案D：内网穿透（临时公网访问）

```bash
# 使用ngrok
ngrok http 8080

# 获得公网链接：http://xxx.ngrok.io
```

**适合：** 临时给客户远程演示

## ❓ 常见问题

**Q: 为什么不像原生App？**
A: 这是PWA技术，需要添加到主屏幕后就像原生App了

**Q: 能发布到App Store吗？**
A: PWA不能直接发布，需要用Cordova打包或开发原生App

**Q: 数据安全吗？**
A: 纯前端演示，数据存在浏览器。正式使用需要后端服务器

**Q: 支持离线使用吗？**
A: 支持！已配置Service Worker，可离线访问
